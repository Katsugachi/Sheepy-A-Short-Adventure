# Sheepy: A-Short-Adventure
A web port of the extremely popular steam game with over 10k+ reviews and a 5 star rating. <br>
The contents are directly from the exported web build (from the original game’s `package.nw`), with no modifications to gameplay or assets — just extracted and hosted for convenient browser access. <br><br>
